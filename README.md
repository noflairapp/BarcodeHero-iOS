# BarcodeHero-iOS

## Formats

#### Supported

- [x] Aztec (native)
- [x] Code 39
- [x] Code 39 mod 43
- [x] Code 128 (native)
- [x] EAN-8
- [x] EAN-13 (ISBN-13, ISSN-13)
- [x] Interleaved 2 of 5
- [x] ITF-14
- [x] PDF417 (native)
- [x] QR (native)
- [x] UPC-E

#### Unsupported

- [ ] Codabar
- [ ] Code 39 Extended
- [ ] Data Matrix
- [ ] MaxiCode
- [ ] RSS-14
- [ ] RSS-Expanded
- [ ] UPC-A
